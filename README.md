# FutureLab

Projeto desenvolvido por Bruno Pulchério, Márcio Alan e Mateus Henrique.
O projeto tem como finalidade o desenvolvimento de um Sistema de Avaliação Docente, no qual coordenadores de instituições de ensino poderão avaliar a qualidade de seus professores e, consequentemente, a qualidade do ensino de sua instituição. O sistema apresentará 3 formas de login - a primeira o acesso é realizado pelo discente, onde, de forma anônima, o mesmo poderá relizar comentários e avaliações sobre os docentes. Também terá a visualização dos perfis docentes, com seus respectivos dados e a visualização de gráficos de desempenho - a segunda opção de login é realizada pelo docente, será possível a edição de seu perfil pessoal e a visualização, tanto das avaliações dos alunos, como dos gráficos de desempenho. - a última forma, o login é efetuado pela coordenação da instituição, será possível a adição, remoção e edição dos docentes conforme necessidade, além das avaliações dos discentes e gráficos de desempenho gerais, semestrais, anuais e etc.
